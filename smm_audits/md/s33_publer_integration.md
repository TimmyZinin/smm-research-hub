# S-33: Publer интеграция

## Гибридная схема

| Канал | Метод | Стоимость |
|-------|-------|-----------|
| Telegram, Bluesky, Mastodon | Прямые API через n8n | Бесплатно |
| LinkedIn, Threads (3 акк), VK, FB, TikTok, IG, YouTube | Publer API | $5/мес + $4/доп. |
| Dev.to, Hashnode | Прямые API | Бесплатно |

## Publer API для n8n

Content Pipeline → Adapter → Publisher → Publer API → платформы.
Publer API документация: https://publer.io/docs/api
