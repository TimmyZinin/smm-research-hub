## 1. Алгоритм и обнаружение

Pocket Casts не имеет алгоритма в стиле Spotify/YouTube. Обнаружение: 70% поиск, 20% курация, 10% трендовое.

### Механизмы обнаружения

| Механизм | Как работает | Приоритет |
| --- | --- | --- |
| Поиск | Индексирует: заголовок, автор, описание, заголовки эпизодов. Точное совпадение > частичное | P0 |
| Трендовое | Скорость подписок за 7-дневное окно. 50 подписчиков за 24ч = трендовое даже с 200 всего | P1 |
| Курированные списки | Редакция (Австралия, Automattic). Тематические коллекции, обновляются периодически | P2 |
| Похожие подкасты | По пересечению подписчиков. Под шоу появляются похожие | P2 |
| Обзор по категории | По RSS <itunes:category>. "Careers" (основная) + "Education" (дополнительная) | P1 |

Скорость подписок — главный рычаг.Каждый новый подписчик непропорционально важен при малой базе. Всплеск 20-30 подписчиков за день = трендовое в категории.

### Ключевой инсайт

Для русскоязычного подкастакросс-промо и прямые ссылки(pca.st/95ecagwv) ценнее алгоритмического подхвата. RU-конкуренция на PC = близка к нулю.

- Фреймворки и чеклисты = закладка + повторный визит
- Конкретные заголовки > общих в 3-5x по поисковым кликам

### Русскоговорящие на PC

- Малый, но высококачественный сегмент: технари, экспат-сообщество
- Билингвальные, комфортно с EN-контентом, ценят RU-опции
- Старшие специалисты (staff-инженеры, лиды, менеджеры)
- Ценят эффективность — короткие, плотные эпизоды
- Уже используют Telegram, LinkedIn, GitHub (целевая СБОРКИ)

### Поведение

- Пики:утренний коммьют (7-9), обед (12-1), вечерняя прогулка (6-8)
- Очередь:эпизоды добавляются осознанно в очередь воспроизведения
- Скорость:1.2-1.5x — норма
- Дослушивание:дослушивают чаще, чем пользователи Spotify
- Низкая терпимость к воде:каждая минута должна заслужить своё место

### Сообщество

- Reddit r/pocketcasts (~50K)
- Блог blog.pocketcasts.com — обзоры подкастов
- Сарафанное радио = основной драйвер роста

### Что НЕ делать на Pocket Casts

- Длинные эмоциональные подводки перед сутью
- Повторять имя/описание шоу в каждом эпизоде
- "Если ты тут впервые..." (для этого есть ep01)
- Просить "подпишись и оставь отзыв" — PC-пользователи считают это снисходительным
