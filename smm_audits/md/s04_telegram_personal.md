## 4. Тон коммуникации — Стиль для Telegram

**Стиль Тима в TG**
- Разговорный— как в чате с другом, не как статья
- Компактные абзацы— связанные мысли вместе, max 4-5 блоков
- Начинай с проблемы аудитории— не с "я сделал"
- Bold для акцентов— 2-3 на пост максимум
- 2-3 эмодзи max— не бусы из смайлов
- CTA дружелюбный— "подписывайтесь, приходите"
- Называй людей по имени— конкретика = доверие
- "Ты/тебе" > "я/мне"— фокус на читателе

**Антипаттерны (ЗАПРЕЩЕНО)**
- "Последний месяц я работал без выходных..."
- Тире (—) между частями предложения
- Двойные отступы между всеми абзацами
- Негативные сравнения ("это не вебинар")
- Пафос ("я вижу вокруг", "и большинство делает")
- Истории "про знакомого"
- Поучительный тон ("приходи, я тебя научу")
- Посты без ссылок и UTM

### Формулы хуков для TG-постов

- **1. Переосмысление** — "Все думают, что [common belief]. На самом деле [surprising truth]." — Переворот ожиданий, провоцирует дочитать.

- **2. Список-зачин** — "3 вещи, которые я понял после [action/event]:" — Конкретика в числе + personal experience = открытие.

- **3. Хук-вопрос** — "Тебе когда-нибудь говорили, что [X]? Вот почему это не работает." — Прямое обращение + вызов.

- **4. Сторителлинг-зачин** — "Вчера тестировал [инструмент/подход]. [Что произошло]. Результат:" — Конкретный эксперимент с outcome.

- **5. Открытые цифры** — "$515 MRR. 6 AI-агентов. 26 платформ. Вот что произошло за последний месяц:" — Цифры в первой строке = attention.

### Форматирование (HTML — единственный формат для Telegram)

```
⛔ ЗАПРЕЩЕНО: **bold**, *italic*, [ссылка](url) — Markdown НЕ работает в Telegram, показывается буквально
⛔ ЗАПРЕЩЕНО: голые URL с UTM в конце поста (некрасиво, нечитаемо)

✅ Жирный: <b>ключевая мысль</b> (2-3 на пост)
✅ Курсив: <i>цитата или внутренний диалог</i>
✅ Код/числа: <code>$515 MRR</code>
✅ Ссылка: <a href="https://timzinin.com?utm_source=telegram&utm_medium=social&utm_campaign=...">подробнее на сайте</a>
   UTM вшит в href, пользователь видит только читаемый текст
✅ Хэштеги: 2-3 max (#AI #агенты #эксперименты)
✅ Emoji: 2-3 max, в начале абзацев для визуальной структуры

Публикация: ТОЛЬКО через ~/auto-publisher/adapters/telegram.py (Bot API, parse_mode: HTML)
⛔ Telethon MCP — ЗАПРЕЩЁН для публикации в канал
```

## 7. Тактики роста

- **Посты, оптимизированные под пересылки** — `Соотношение пересылок к просмотрам: 15-25% = вирусный пост` — Писать посты, которые хочется переслать: полезные списки, острые мнения, "посмотри что написал", личные провалы с уроками. Каждая пересылка = бесплатное привлечение.

- **Кросс-промо с каналами** — `Обмен упоминаниями: 5-15% конверсия от пересечения аудитории` — Найти 5-10 каналов в нише (AI, карьера, tech, 500-5K subs). Написать админам: "Давай сделаем взаимный пост — я расскажу о твоём канале, ты о моём." 1-2 свопа в неделю.

- **Привлечение через бот** — `@Sborka_work_bot → подписка на канал` — В боте @Sborka_work_bot добавить кнопку "Канал Тима" или автопредложение подписки после приветствия. Каждый новый пользователь бота = потенциальный подписчик канала.

- **Участие в группах** — `Активность в 5-10 тематических группах с 1K+ участниками` — Писать полезные ответы в чатах (Botanica, career groups, AI groups). В профиле — ссылка на канал. НЕ спамить, а давать ценность → любопытство → переход на канал.

- **Регистрация в каталогах** — `tgstat.ru + telemetr.io + combot.org` — Зарегистрировать канал во всех крупных каталогах. Бесплатно. Улучшает обнаруживаемость и даёт социальное доказательство (рейтинги, статистика).

- **Стратегия инвайт-ссылок** — `Уникальные инвайт-ссылки для каждого канала/платформы` — Создать отдельные инвайт-ссылки для: LinkedIn bio, Instagram bio, описание YouTube, sborka.work, подпись email. Отслеживать какой канал приводит подписчиков.

- **Событийный рост** — `Вебинары СБОРКИ → подписчики канала` — "Подписывайтесь на канал для анонсов следующих вебинаров и эксклюзивных материалов". Каждый вебинар = 10-30 новых подписчиков канала при правильном CTA.
