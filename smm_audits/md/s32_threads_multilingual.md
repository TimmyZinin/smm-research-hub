# S-32: Threads мультиязычная стратегия

## 3 аккаунта для старта

| Аккаунт | Язык | Частота | Instagram |
|---------|------|---------|-----------|
| @timzinin (существующий) | RU | 5+/день | Существующий IG |
| Новый EN | EN | 5+/день | Новый IG (отдельный email) |
| Новый TH | TH | 5+/день | Новый IG (отдельный email) |

## Почему отдельные аккаунты

Смешивание языков в одном аккаунте = алгоритм плохо классифицирует аудиторию, охваты размываются. Отдельный Threads = отдельный Instagram = отдельный email.

## Контент

Адаптация из RU-нарратива через MiniMax M2.5:
- RU → EN: прямой перевод + локализация примеров
- RU → TH: перевод + адаптация культурного контекста

Все аккаунты публикуют одинаковый набор тем (AI-находки), но адаптированный под язык и культуру.

## Публикация

Через Publer API (все 3 аккаунта подключены к Publer).
