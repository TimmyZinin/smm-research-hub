# S-10 — VC.ru

> Главная русскоязычная бизнес-медиаплатформа: лонгриды, карма, комментарии

## 1. Алгоритм и ранжирование VC.ru

VC.ru использует гибридную систему: алгоритм + редакция. Engagement velocity в первые 1-2 часа = ключевой фактор.

#### 5 стадий дистрибуции статьи

```
Стадия 1 (0-30 мин):   ~200-500 юзеров (подписчики + подсайт)
Стадия 2 (30м-2ч):     3+ плюсов, 2+ комментов → ~2,000-5,000 в ленте
Стадия 3 (2-6ч):       10+ плюсов, 5+ комментов → "Популярное" → 10K-50K
Стадия 4 (6-24ч):      Топ → Карусель "В тренде" (100,000+ просмотров)
Стадия 5 (Выбор ред.): Закреплена на главной 24-72ч → 50K-500K+ просмотров
```

| Фактор | Вес | Описание |
| --- | --- | --- |
| Скорость вовлечённости | ВЫСОКИЙ | Плюсы, комменты, закладки в первые 1-2 часа |
| Количество комментариев | ВЫСОКИЙ | 10+ комментов за 2ч = серьёзный алгоритмический подъём |
| Глубина прочтения | СРЕДНЕ-ВЫСОКИЙ | Глубина прокрутки + время на странице. Прочитали >60% = преференция |
| Карма автора | СРЕДНИЙ | Выше карма = шире начальный пул показа |
| Релевантность подсайта | СРЕДНИЙ | Правильный подсайт = лучшая тематическая дистрибуция |
| Выбор редакции | МАКС | Ручной выбор редакции → гарантированная главная |

## 4. Конкурентный анализ

> **Инсайт:** Голубой океан:Пересечение "AI-агенты + карьера + строительство на публике" практически не занято на VC.ru. "One Brain, Many Agents" как карьерный фреймворк — уникальный угол, который работает через все подсайты.

**** (3-5x/неделю1.5K-8.8K views/статьяAI + запуски продуктов)
  Стратегия:Новостная + практические обзоры инструментов. Комбинирует агрегацию новостей с "как вы можете использовать это".Слабость:Нет личного брендового сторителлинга, нет карьерного угла.

**** (236+ комментов на flagshipМаркетинговые кейсыСильная вовлечённость в комментах)
  Стратегия:Длинные кейс-стади с бизнес-метриками. Сочетание культурных инсайтов с цифрами.Тактика для заимствования:Стабильность + глубокие комменты → рост с 500 до 3K за 12 мес.

**** (AI-автоматизация0→2K за 6 месШаблоны + рабочие процессы)
  Стратегия:Практические AI-рабочие процессы для solo-предпринимателей. "Как делать больше с меньшим" через AI.Ближайший конкурент, но без карьерного угла и без личного бренда масштаба Тима.

**** (774+ лайковОптимизация конверсииФреймворки + примеры)
  Стратегия:Техническая глубина без жаргона, практичные чеклисты.Тактика:Фреймворки = контент для закладок = шеринг.

**** (349+ лайковРегуляторика/комплаенсНишевая экспертиза)
  Вывод:Узкая экспертиза = меньше конкуренции, высокая ценность. Нишевые темы = лояльная аудитория.

## 5. Стратегия роста через комментарии

> **Инсайт:** Главная стратегия для VC.ru:Комментарии = рычаг роста #1. VC.ru — платформа с акцентом на комментарии. Экспертные комменты строят узнаваемость имени, поднимают карму и привлекают подписчиков быстрее, чем статьи на старте.

```
5-10 комментов/день × 30 дней = 150-300 точек контакта
Среднее 2-3 upvotes/коммент = +300-900 кармы за месяц
Цель: из +7 → +100-200 за 30 дней через комменты
```

## 6. Tone of Voice для VC.ru

> **Внимание:** VC.ru community аллергична к:явной рекламе, пустым обещаниям, инфобизнес-тону, hype без substance, "я вижу вокруг" мессианским повествованиям. Аудитория: скептичные профессионалы, которые ценят данные и личный опыт.

Что работаетКейсы на основе данных"Мы протестировали 14 AI-моделей для видео. Вот бенчмарки: [таблица с ценами, скоростью, качеством]."Честные провалы"Я потратил 3 месяца на [X] и это не сработало. ## 7. CTA-стратегия: VC.ru → sborka.work

> **Внимание:** Антирекламная культура VC.ru:Явная самореклама получает минусы и может убить карму. Мягкий CTA = единственный рабочий подход. Правило: 80%+ полезного контента, <20% с любым упоминанием продукта.

> **Инсайт:** Лучший CTA-паттерн для VC.ru:Статья с реальными данными → заканчивается "если хотите больше таких данных, у нас есть [TG-канал / СБОРКА]." CTR мягкого CTA: 3-8% vs <1% для жёсткой рекламы.

## 8. Паттерны вирусного контента

#### Что набирает 50K+ views на VC.ru

AI-угроза + влияние на карьеру"Вы потеряете работу из-за AI" фактор страха + "но вот как выжить" решение. Пример: "Почему AI-агенты заменят 40% junior-позиций (и как остаться востребованным)".

Практические AI-инструменты с личным тестомНе список инструментов, а "я протестировал X месяц и вот результаты с цифрами". Личный опыт + бенчмарки = золото.

Провалы стартапов + уроки"Мы заплатили $50K за [сервис], сэкономили $30K используя AI." ROI + конкретные цифры = вирус.

Данные по зарплатам + карьерные переходы"Реальные зарплаты в AI: от $3K до $30K." Данные по зарплатам = машина вовлечённости на VC.ru.

#### Формулы заголовков

```
1. "[Number] [keyword] для [audience] в 2026"
   → "7 AI-навыков которые гарантируют зарплату 300К+"

2. "Я [action] [timeframe]. Вот что случилось."
   → "Я автоматизировал всю работу AI-агентами за 30 дней."

3. "Почему [mainstream belief] — ошибка"
   → "Почему 90% AI-пилотов провалятся"

4. "Как мы [achieved result] с [specific tool/method]"
   → "Как мы сократили time-to-hire на 60% через AI"

5. "Полный гайд по [topic]: [year] edition"
   → "Полный гайд по AI-карьере: 2026 edition"
```
