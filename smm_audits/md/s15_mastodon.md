# S-15: Mastodon

> Fediverse · ActivityPub · Нулевой алгоритм · Анти-коммерческая культура

## 1. Механика Fediverse и дистрибуция контента

Mastodon — крупнейшая платформа в Fediverse — сети взаимосвязанных, независимо управляемых серверов, общающихся через открытый протоколActivityPub. ~10-12M активных пользователей ежемесячно в совокупности, из них 5-7M на Mastodon. Пользовательская база непропорционально состоит из разработчиков, учёных и сторонников open source —именноцелевая аудитория Тима.

### Три ленты

| Лента | Контент | Охват |
| --- | --- | --- |
| Домашняя | Посты от ваших подписок (с любого инстанса) | Только ваши подписчики |
| Локальная | Все публичные посты вашего инстанса | Все на mastodon.social (~2,2M аккаунтов) |
| Федеративная | Все публичные посты, о которых знает ваш инстанс | Варьируется — зависит от связей инстанса |

```
КАК ПОСТ ДОСТИГАЕТ НОВЫХ ЛЮДЕЙ:
  1. Прямые подписчики → Домашняя лента
  2. Бусты → ВСЕ подписчики бустера видят его (ОСНОВНОЙ ВИРУСНЫЙ МЕХАНИЗМ)
  3. Хештеги → Поиск + подписка с версии Mastodon 4.x
  4. Ретрансляционные серверы → Расширяют федеративную ленту
  5. Обзор/Тренды → Курируется админами инстанса (по желанию)

  КЛЮЧЕВОЙ ИНСАЙТ: Получить буст от аккаунтов с большим количеством
  подписчиков на активных инстансах — ГЛАВНЫЙ рычаг роста.
```

### Mastodon vs Bluesky vs Twitter

| Функция | Mastodon | Bluesky | Twitter/X |
| --- | --- | --- | --- |
| Алгоритм | Нет (хронология) | Хронология + кастомные ленты | Алгоритмический + Для вас |
| Доступ к API | Бесплатный, открытый | Бесплатный, открытый | $100-$42K/месяц |
| Лимит символов | 500 | 300 | 280 (бесплатно) |
| Федерация | Полная (ActivityPub) | AT Protocol | Нет |
| Редактирование постов | Да | Нет | Только Premium |
| Реклама | Нет | Нет | Повсюду |

> **Инсайт:** Преимущество API:Бесплатный открытый API Mastodon — огромный плюс vs $200/месяц у Twitter. У Тима уже зарегистрировано приложение «Zinin Corp SMM» с 6 опубликованными статусами. Автопубликация стоит $0.

## 3. Анализ конкурентов

**** ()
  Обзоры LLM-инструментов, фрагменты Python, ссылки на блог. Очень высокое соотношение бустов к лайкам. Треды о новых AI-инструментах получают 200-500 бустов.Урок:искренний процесс исследования, всегда обучающий, никогда рекламный.

**** ()
  Анти-энштификация, техно-политика. 500-2000 бустов на длинных тредах. Использует Mastodon как ОСНОВНУЮ платформу (не кросс-пост).Урок:мастерство тредов + аутентичность массово вознаграждаются.

**** ()
  Этика AI, критика индустрии. Спорные позиции получают максимальную вовлечённость.Урок:аутентичный голос и принципиальные позиции побеждают.

**** ()
  DevOps/Ansible, Raspberry Pi. Фото аппаратных проектов получают больше всего бустов.Урок:всегда даёт ценность в самом посте, никогда просто «новое видео, посмотрите.»

**** ()
  Технические зины/инфографика. Визуальный контент получает 500-1500 бустов.Урок:визуальный технический контент — король на Mastodon. Сделать сложные темы доступными через визуализацию.

## 4. Культура сообщества и доступность

Культура Mastodon сформирована анти-надзорным капитализмом, open source этикой и принципами доступности. Понимание этогобезоговорочнонеобходимо для успеха.

> **Внимание:** Ключевые ценности:Анти-надзорный капитализм— любой «хак роста» или «воронка» в языке = враждебностьOpen source и децентрализация— вклад в FOSS приносит огромное уважениеДоступность не опциональна— alt-текст, CamelCase хештеги, предупреждения о контенте = базовая вежливостьСообщество важнее знаменитостей— никто не хочет быть «инфлюенсером Mastodon»Анти-метрическая одержимость— праздновать количество подписчиков не принято

## 5. Стратегия хештегов

Хештеги — основной механизм обнаружения на Mastodon. Пользователи могут ПОДПИСЫВАТЬСЯ на хештеги (с версии 4.x), и они появляются в домашней ленте. Используйте 3-5 на пост, максимум 7. Больше = спам.

| Уровень | Хештеги | Примерные подписчики |
| --- | --- | --- |
| Уровень 1 (большинство постов) | #AI, #ArtificialIntelligence, #MachineLearning, #OpenSource, #Python, #Tech | 40K-100K+ каждый |
| Уровень 2 (ротация) | #AIAgents, #MultiAgent, #AgenticAI, #BuildInPublic, #DevOps, #TechCareers, #Fediverse | 5K-40K каждый |
| Уровень 3 (конкретные) | #CrewAI, #LangChain, #Docker, #SelfHosted, #FOSS, #NLP, #RAG | 1K-15K каждый |

```
ФОРМУЛЫ ХЕШТЕГОВ ПО КОНТЕНТНЫМ СТОЛПАМ:

  Сборка на публику: #BuildInPublic #AI #AIAgents #Python #Automation
```

## 8. Кросс-платформенность: федерация ActivityPub

Протокол ActivityPub Mastodon обеспечивает совместимость с растущей экосистемой — уникальное преимущество, которого нет ни у одной централизованной платформы.

| Платформа | Тип | Совместимость | Приоритет |
| --- | --- | --- | --- |
| WordPress (плагин ActivityPub) | Блог | Посты блога появляются как посты Mastodon, получают комментарии | ВЫСОКИЙ |
| Lemmy | Reddit-подобный | Подписка на сообщества из Mastodon, просмотр постов | СРЕДНИЙ |
| PeerTube | YouTube-подобный | Подписка на каналы, просмотр видеопостов | СРЕДНИЙ |
| Threads | Микроблог | Ограниченная поддержка ActivityPub с декабря 2024, по желанию | НИЗКИЙ |
| Pixelfed | Instagram-подобный | Полная совместимость: подписка, комментарии, бусты | НИЗКИЙ |
| Flipboard | Новости | Поддержка ActivityPub с 2024 | НИЗКИЙ |

> **Инсайт:** Мощный ход:Если sborka.work добавит поддержку ActivityPub (плагин WordPress или кастомная реализация), каждый исследовательский отчёт автоматически федерируется в Mastodon. Пользователи подписываются на@blog@sborka.workи видят статьи в своей ленте. Это самый мощный кросс-платформенный CTA — обнаружение контента без какого-либо продвижения.

```
ПРАВИЛА КРОСС-ПОСТИНГА:
  ЛУЧШЕ: Писать для Mastodon сначала → адаптировать для других платформ
  ХОРОШО: Опубликовать на Mastodon, затем расшарить адаптированную версию
  НОРМАЛЬНО: Написать LinkedIn-пост, затем ИСКРЕННЕ переписать для Mastodon
  ПЛОХО: Авто-кросс-пост из Twitter в формате "RT"
  ПЛОХО: Опубликовать идентичный текст на Mastodon и LinkedIn
  ПЛОХО: Ссылка на свой LinkedIn/Twitter пост ("колониализм Fediverse")
```
