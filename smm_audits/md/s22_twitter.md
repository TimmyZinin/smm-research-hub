# Twitter / X

> S-22 — Главная платформа для Tech/AI аудитории строителей

## 1. Алгоритм и распространение контента

### For You vs Подписки

For You (алгоритмическая)— вкладка по умолчанию, где происходит виральность. 50% контент от подписок (ранжированный моделью Real Graph) + 50% вне сети (через Social Graph + SimClusters ~145K сообществ по интересам).

Following (хронологическая)— чистая хронология. ~15-20% сессий. Важна для лояльных подписчиков.

### Сигналы вовлечённости и их веса

| Сигнал | Вес | Примечание |
| --- | --- | --- |
| Закладка | ~4x | САМЫЙ мощный сигнал. «Хочу вернуться к этому» |
| Ответ | ~1x | Начало разговора = высокое качество |
| Ретвит | ~1x | Сигнал усиления |

## 2. Форматы контента

### Одиночные твиты — 4 формата

### Треды — самый мощный формат

Генерируют наивысшую суммарную вовлечённость. Оптимум:5-10 твитов. Первый твит-крючок определяет 90% охвата. Основные твиты — самостоятельно ценные. Завершение: CTA или вопрос на вовлечение.

### Видео — 3-5x алгоритмический буст в 2025-2026

Twitter активно бустит видео (конкуренция с TikTok/Shorts). Запись экрана 30-90 сек, говорящая голова + экран 2-5 мин, демо-ролики 15-30 сек. Существующий AI-видео пайплайн (shorts_v2.py, $0.09/мин) → почти нулевая стоимость перепрофилирования.

### Другие форматы

| Формат | Применение |
| --- | --- |
| Spaces (аудио) | Еженедельный «Отчёт агентов» 30 мин + приглашённые строители. Фиолетовое уведомление = гарантированная видимость |
| Сообщества | «AI Builders», «Build in Public», «Indie Hackers» — пост 1-2 раза/неделю |
| Опросы | 4 варианта, 24ч. Безбарьерная вовлечённость. Далее анализ результатов |
| Статьи (Premium) | Длинные гайды, бенчмарки. Постоянный URL для шаринга, индексируется Google |

## 4. Культура Tech Twitter

### Идентичность строителя = высший статус

«Я это запустил» весит больше любых регалий. Каждый твит должен подтверждать, что Тим СТРОИТ. Скриншоты > описания. Вывод терминала > обещания. Работающие URL > питч-деки.

### Культура цитирования

| Тип цитирования | Применение | Для Тима |
| --- | --- | --- |
| Поддержка (добавляя ценность) | «Отличное замечание. По моему опыту с агентами: [инсайт]» | ОСНОВНОЙ |
| Несогласие (уважительное) | «Мои данные показывают обратное. [доказательство]» | OK когда обосновано |
| Унижение (враждебное) | Насмешка над оригиналом | НИКОГДА |

### Правило «10 ответов перед 1 постом»

Каждый день: 10 продуманных ответов на твиты из ниши → потом свой контент. Это: (a) показ профиля чужим аудиториям, (b) отношения с авторами, (c) сигнал для алгоритма что ты активный участник.

## 5. Tone of Voice — адаптация для Twitter

### Twitter вознаграждает скорость мнений

LinkedIn = polished thought leadership. Telegram = conversational warmth. Twitter =fast, sharp, opinionated takes backed by evidence. Контент должен быть dialogical — argue, debate, challenge, respond.

### Голос дневника строителя — аутентичный голос Тима в Twitter

```
Day 47 of running AI bloggers in production.

Today's surprise: the Threads API rate-limited us after 247 posts.
Docs say 250/day. Reality: 247.

Workaround: stagger posts across 2 tokens.
Cost of the fix: 12 minutes of my time.

These are the details nobody writes about.
```

### Адаптации от существующего тона коммуникации

| Правило | Адаптация для Twitter |
| --- | --- |
| Не начинай с «я сделал» | Twitter: «I» OK когда = доказательство работы («I shipped», «I tested», «I benchmarked») |
| Без мессианских нарративов | СОХРАНИТЬ — детектор кринжа на Twitter ещё выше |
| Ссылки в каждом посте | ИЗМЕНИТЬ — ссылки в ОТВЕТАХ, не в основном твите (на 40-60% меньше показов) |
| Дружелюбный тон | ЗАОСТРИТЬ — Twitter «дружелюбный» более расслабленный + допускает резкость |

### Что НИКОГДА не писать в Twitter

- "I'm so excited to announce..." (cringe)
- "Thread incoming!" без самого thread
- "Please RT" / "share this" (begging = credibility kill)
- LinkedIn-style длинные абзацы с эмодзи
- "AI will take all jobs" doom (clash с SBORKA mission)

## 9. Кросс-платформенная синдикация

### Фреймворк «Одна идея, пять форматов»

```
Одна идея ("AI agents cost $47/mo") →
1. Twitter: Hot take single tweet
2. LinkedIn: Professional breakdown (500 words)
3. Threads: Micro-take (500 chars)
4. Telegram: Разговорный пост (RU)
5. Dev.to: Full technical article (2000+ words)
```

### Правила адаптации

| Направление | Правило |
| --- | --- |
| Twitter → LinkedIn | Expand, add context, remove edge/sharpness, add "takeaway" framing |
| LinkedIn → Twitter | Extract hottest take, strip corporate tone, add numbers |
| Twitter → Threads | Compress to 500 chars, keep hook, add emoji sparingly |
| Twitter → Telegram | Translate to RU, add warmth, expand context |
| YouTube → Twitter | 30-sec clips from videos, screen recordings, key moments |

> **Внимание:** Что НЕ кросс-постить:Горячие мнения Twitter → LinkedIn (слишком резко). Эссе LinkedIn → Twitter дословно (слишком отполировано). Платформо-специфичные ловушки вовлечённости (опросы для одной платформы).
