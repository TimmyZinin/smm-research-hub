## 1. Алгоритм и обнаружение

### 1.1 Как Spotify рекомендует подкасты (2025-2026)

Три ключевых системы:Коллаборативная фильтрация(основной сигнал — слушатели похожих подкастов),Контентные сигналы(NLP заголовок/описание + автоматическая транскрипция + категории),Метрики вовлечённости.

### 1.2 Метрики вовлечённости (по приоритету)

| # | Метрика | Значение |
| --- | --- | --- |
| 1 | Процент дослушивания | Самый важный сигнал. >60% = качество |
| 2 | Подписка после прослушивания | Сколько слушателей подписываются после пробы |
| 3 | Процент возвратов | Как часто подписчики возвращаются |
| 4 | Сохранения/репосты | Эпизоды сохранённые/расшаренные |
| 5 | Процент запусков | CTR от показов к прослушиванию |
| 6 | Серия прослушиваний | Бинж-сигнал (подряд несколько эпизодов) |

### 1.3 Поверхности обнаружения Spotify

| Поверхность | Как работает | Приоритет для СБОРКИ |
| --- | --- | --- |
| Главная лента | Персонализированные карточки | ВЫСОКИЙ |
| «Шоу, которые могут понравиться» | Коллаборативная фильтрация | ВЫСОКИЙ |
| Результаты поиска | Заголовок, описание, транскрипты | ВЫСОКИЙ |
| Обзор категорий | Редакторские + алгоритмические | СРЕДНИЙ |
| «Свежие находки» для подкастов | Новые шоу с сильной вовлечённостью | ВЫСОКИЙ (4 эпизода = ещё подходит) |

### 1.4 Поисковое ранжирование

1. Точное совпадение в заголовке шоу— максимальный вес. "СБОРКА" → #1 для "сборка подкаст"
2. Ключевые слова в описании— второй вес
3. Ключевые слова в заголовках эпизодов— индексируются
4. Автоматические транскрипты— с 2025 Spotify индексирует аудио
5. Вовлечённость— высоко-вовлечённые шоу ранжируются выше
6. Свежесть— недавно обновлённые шоу → выше

Триггеры для редакторского внимания:Регулярная публикация, >50% среднее дослушивание, >10% еженедельный рост подписчиков, упоминания в соцсетях со Spotify-ссылками.

## 2. Форматы контента

### 2.1 Типы эпизодов: Эффективность

| Формат | Среднее дослушивание | Обнаружение | Рекомендация |
| --- | --- | --- | --- |
| Соло-монолог(текущий) | 55-65% | Среднее | Основной формат |
| Интервью/гость | 45-60% | ВЫСОКОЕ | 1x/мес для кросс-промо |
| Q&A от слушателей | 60-70% | НИЗКОЕ | Для удержания |
| Мини-эпизод(5-8 мин) | 75-85% | Среднее | Для алгоритмического буста |
| Трейлер(1-2 мин) | 90%+ | ВЫСОКОЕ | Создать немедленно |

### 2.2 Видео-подкасты на Spotify

Видео = 2-3x больше показовчем только аудио. Видео-эпизоды показываются с визуальными превью в ленте (значительно выше CTR).

| Параметр | Требование |
| --- | --- |
| Разрешение | Мин. 1080p (1920×1080), 16:9 |
| Формат | MP4 (H.264 video, AAC audio) |
| Загрузка | Через Spotify for Podcasters (не RSS) |
| Макс. размер | 4GB / эпизод |
| Субтитры | Авто-генерация, лучше ручная (SRT) |

Стратегия для СБОРКИ:Presentation-style (слайды + voiceover) — просто в производстве через ffmpeg. Видео shorts из Lisa pipeline можно репурпозить.

### 2.3 Интерактивные функции

- Опросы— карточки в плеере. Буст вовлечённости. "Что для тебя самое сложное в поиске работы?"
- Вопросы и ответы— слушатели задают вопросы через страницу эпизода → контент для будущих эпизодов
- Главы— список секций в плеере. Снижает отвал
- Транскрипты— индексируются для поиска. Загрузить ручные для точности на русском

### 2.4 Чек-лист на каждый эпизод

- ☐ Заголовок содержит primary keyword в первых 5 словах
- ☐ Описание включает 3-5 keywords естественно
- ☐ Chapter markers проставлены (intro, content, tips, CTA)
- ☐ Транскрипт загружен (из скрипта)
- ☐ Poll или Q&A добавлен
- ☐ Ссылки в show notes с UTM
- ☐ "Follow" CTA произнесён на 2-3 минуте и в конце

### 3.3 Конкурентные преимущества СБОРКИ

- AI-ракурс— мало карьерных подкастов про AI в поиске работы
- Короткий формат— 14-15 мин идеален для процента дослушивания (конкуренты 30+ мин)
- Интегрированная экосистема— бот + сайт + подкаст + сообщество сильнее одиночного продукта
- Двуязычный потенциал— могут обслуживать и RU и EN аудитории

### 3.4 Пробелы для использования

1. Видео-подкасты— почти никто из RU-карьерных не делает видео на Spotify
2. AI-инструменты поиска работы— никто не учит практическому AI-для-карьеры в RU-подкастах
3. Мини-эпизоды— нет RU-карьерных подкастов с 3-5 мин быстрыми советами
4. Интерактивные функции— опросы, Q&A практически не используются
5. Клипы для соцсетей— RU-подкастеры редко создают короткие клипы для соцсетей

## 5. Tone of Voice для Spotify

### 5.1 Гайд по стилю аудио

- Зацепка в начале (первые 15 сек) — КРИТИЧЕСКИ ВАЖНО.НЕ "Привет, это Сборка" как первые слова
- Паттерн:[ЗАЦЕПКА] → [ДЖИНГЛ 5 сек] → [Стандартное интро] → [Контент]
- Пример:"Ты отправил 50 откликов и ни одного ответа. Знакомо? Сегодня разберёмся, почему." [ДЖИНГЛ] "Привет, это Сборка..."

### 5.2 Spotify-специфика

| Правило | Реализация |
| --- | --- |
| Разговорное «ты» | Один человек, не аудитория. "Если ты сейчас ищешь..." |
| Голосовые ориентиры | "Первое.", "Второе.", "Главная мысль:", "Короче говоря:" |
| Без клише | Нет "жми подписку", нет фейковой энергии |
| Голос CTA | Мягко, не навязчиво: "Если тебе полезно, нажми 'Подписаться'" |
| Качество аудио | -16 LUFS, 44.1 kHz, 128-192 kbps CBR, макс 2 сек тишины |

### 5.3 Двуязычная стратегия

Основная серия:Русский.Ежемесячный спецвыпуск:1 англоязычный эпизод ("СБОРКА International: [Тема]"). Описание шоу: RU + EN секция.
