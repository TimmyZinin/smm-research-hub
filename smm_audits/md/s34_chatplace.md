# S-34: ChatPlace.io — воронка подписки

## Механика

1. Видео (TikTok/Reels/Shorts) → рассказ о AI-находке
2. CTA: "Напиши АВТОМАТИЗАЦИЯ в комментариях"
3. ChatPlace бот проверяет подписку
4. Если подписан → отправляет ссылку на GitHub-репо
5. Если не подписан → просит подписаться

## Платформы

Instagram, TikTok, YouTube

## Результат

Рост подписчиков + engagement + лид в CRM
