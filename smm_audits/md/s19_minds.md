# S-19: Minds.com

> Открытая соцсеть с токен-экономикой — MINDS ERC-20 бусты, прозрачный алгоритм, крипто-нативная аудитория

## 1. Алгоритм и распространение контента

Minds использует принципиально другую модель:никакого непрозрачного чёрного ящика. Платформа с открытым исходным кодом (GitLab:gitlab.com/minds), и алгоритм ранжирования доступен для публичного аудита.

### Три механизма ленты

### Факторы распространения

| Фактор | Влияние | Действие для Тима |
| --- | --- | --- |
| Ранняя вовлечённость (первые 2ч) | Высокое — запускает широкое распространение | Публиковать когда аудитория активна |
| Расходы на бусты | Очень высокое — главный рычаг роста | 50-100 токенов/неделю на лучшие посты |
| Reminds (репосты) | Высокое — эквивалент ретвитов | Создавать фреймворки и горячие мнения для распространения |
| Комментарии | Высокое — повышает ранжирование + сообщество | Заканчивать посты вопросами, отвечать на каждый комментарий |
| Хэштеги | Среднее — механизм обнаружения | 3-5 целевых: #AI #MultiAgent #Automation #Crypto |

## 2. Токен-экономика MINDS

### Обзор токена

| Свойство | Детали |
| --- | --- |
| Тип | ERC-20 (Ethereum mainnet) |
| Предложение | 12 миллиардов (фиксированный лимит, без инфляции) |
| Диапазон цен | ~$0.01-0.05 (волатильно) |
| Заработок | Ежедневные награды за вовлечённость, чаевые Wire, платежи Supermind, рефералы |
| Расходы | Буст постов, взаимный буст, чаевые Wire, подписка Minds Pro |

### Токен-стратегия для Тима

```
Экономика бустов:~1 токен MINDS = 1,000 показов| 30 токенов = 30K просмотров лучшего поста | Общая стоимость: ~$0.30-1.50
```

| Уровень | Стоимость | Функции | Рекомендация |
| --- | --- | --- | --- |
| Бесплатный | $0 | Полный постинг, бусты, группы, чаевые Wire | Начать здесь (Месяц 1) |
| Minds+ | $7/мес | Значок верификации, дополнительный охват, кастомные темы | Оценить на Месяц 2-3 |
| Pro | $60/мес | Кастомный домен, подписки, платные стены, стримы, аналитика | Только при 500+ подписчиках |

### Потоки дохода на Minds

1. Wire-подписки— регулярные токен-платежи от подписчиков (аналог Patreon)
2. Supermind— платные Q&A: установить минимум в 5-10 MINDS, позиционировать как «Спроси меня о мультиагентном AI»
3. Платный контент— эксклюзивные чертежи AI-агентов за Wire-стеной
4. Ежедневные награды за вовлечённость— пассивный заработок токенов от активного постинга
5. Реферальные бонусы— токены за каждого нового зарегистрированного пользователя

> **Инсайт:** Реалистичный доход при зрелости (6+ месяцев, 1000+ подписчиков):$70-300/месяцв токенах MINDS. Не основной источник дохода, но покрывает затраты на бусты и доказывает спрос. Реальная ценность = привлечение крипто-грамотной аудитории к сервисам sborka.work.
