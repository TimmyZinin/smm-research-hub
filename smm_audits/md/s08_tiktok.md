## 1. Алгоритм TikTok 2026

### 3-этапная воронка FYP

```
Этап 1: Малая выборка (100-300 просмотров) → тестирование
Этап 2: Средняя выборка (1K-10K) → если вовлечённость высокая
Этап 3: Виральный пуш (10K-1M+) → если продолжает расти
Каждый этап = новая оценка сигналов вовлечённости
```

### Сигналы ранжирования (по приоритету)

| # | Сигнал | Вес | Что изменилось с 2024 |
| --- | --- | --- | --- |
| 1 | Время просмотра / Процент досмотра | Наивысший | 70%+ досмотра нужен для виральности (было 50%) |
| 2 | Репосты | Очень высокий | Теперь сигнал #2 (было ниже). "Отправить другу" = главная метрика |
| 3 | Повторные просмотры | Высокий | Контент, который хочется пересмотреть, продвигается агрессивно |
| 4 | Сохранения | Высокий | Сохранение = сигнал ценности для алгоритма |
| 5 | Комментарии | Средний | Скорость комментариев в первые 60 мин |
| 6 | Лайки | Средний | Наименее важный из сигналов вовлечённости |

### Новое в 2025-2026

| Изменение | Влияние |
| --- | --- |
| Продвижение длинного контента (3-10 мин) | Больше не только Shorts. Обучающие видео, глубокие разборы получают пуш |
| TikTok SEO повзрослел | 41% поколения Z используют TikTok как поисковик. Ключевые слова в описаниях критичны |
| Оценка оригинальности | Оригинальный контент получает буст. Репосты/дубликаты понижаются |
| Система тематического авторитета | Алгоритм строит "экспертный профиль". Стабильный постинг в одной нише = буст |

> **Инсайт:** Ключевой вывод:Время просмотра > всё остальное. Короткие видео (11-18 сек) с 70%+ досмотра = самый надёжный путь в FYP. Репосты стали сигналом #2 — контент "пришли другу" = рычаг роста.

## 2. Смена ниши: Путешествия → AI/Карьера

### 3 варианта перехода

| Вариант | Потеря подписчиков | Срок реклассификации | Рекомендация |
| --- | --- | --- | --- |
| A: Постепенный мост | ~5% | 3-4 недели | Рекомендуем |
| B: Резкий переход | 10-20% | 2-3 недели | Быстрее, но болезненнее |
| C: Новый аккаунт | 100% (с нуля) | 1-2 недели | Только если аккаунт в теневом бане |

### 4-недельный план перехода

- **Подготовка**: Обновление описания, идеи для связующего контента, пакетная запись
- **Нед. 1-2**: Мост: Путешествия + Карьера (цифровой кочевник, удалёнка, фриланс-AI)
- **Нед. 3-4**: Полный переход: 100% контент об AI/Карьере
- **Нед. 5+**: Масштабирование: удвоить успешные форматы

- **Идеи связующего контента (Путешествия → Карьера)** — "Как я работаю удалённо из Турции с помощью AI-инструментов" | "Карьера цифрового кочевника: как я построил AI-бизнес за границей" | "От тревел-блогера до архитектора AI-агентов — моя история" | "5 AI-инструментов которые позволяют работать откуда угодно"

> **Внимание:** Сроки реклассификации:10-15 видео для начального тестирования алгоритмом → 20-30 видео для значительного сдвига → 50+ для полной реклассификации. При 2 видео/день = ~25 дней до полной перестройки.

## 3. Анализ конкурентов

**@tiktokai_news / Агрегаторы AI-новостей** (Быстрый рост | 3-5 видео/день | Запись экрана + текст | Скорость = оружие)
  Вывод:Скорость реакции на AI-новости. Первые 1-2 часа после анонса = максимальный пуш в FYP.

**@aliabdaal / Продуктивность + Карьера** (Огромная аудитория | Говорящая голова | Советы для учёбы/карьеры | Высокое качество продакшена)
  Вывод:Говорящая голова с текстовым наложением — лучший формат. Искренняя подача > отполированный продакшен. На 31% выше вовлечённость для неотрепетированного.

**@careercontessa / Карьерные советы** (Чистая карьерная ниша | "POV: тебя только что взяли на работу" | Переходы | Узнаваемые сценарии)
  Вывод:Формат POV для карьерных сценариев = виральный формат. "POV: твоё резюме прошло ATS" с эффектом перехода.

**Русскоязычные AI/Tech-создатели** (Маленькая аудитория (1-5K) | Нерегулярный постинг | Низкая конкуренция)
  Вывод:Огромный пробел.Русскоязычных создателей AI+Карьера на TikTok практически нет. Преимущество первого игрока. 60% контента на RU, 40% EN.

**Русскоязычные карьерные коучи** (В основном на YouTube/Instagram | Слабое присутствие на TikTok | Длинный формат)
  Вывод:Карьерные коучи не освоили формат TikTok (коротко, динамично). Тим может быть первым, кто адаптирует карьерные советы под TikTok.

## 6. Тон коммуникации

**РАБОТАЕТ**
- Живой, аутентичный, неотрепетированный стиль (на 31% выше вовлечённость)
- Быстрый темп речи, без пауз
- Спорные мнения с данными
- "Я построил это / я помог 50 людям" (на основе опыта)
- Двуязычность: 60% RU / 40% EN
- Самоирония
- Зрительный контакт с камерой (говорящая голова)

**НЕ РАБОТАЕТ**
- Перепродюсированное, студийное качество (выглядит как реклама)
- Длинные интро (3+ сек до хука = прокрутка)
- Общая мотивация без конкретики
- "Подпишись" (штрафуется)
- Водяные знаки других платформ
- Горизонтальное видео (только 9:16)
