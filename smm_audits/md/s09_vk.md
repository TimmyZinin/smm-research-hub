## 1. Алгоритм VK Smart Feed 2026

VK использует ML-based "Smart Feed" — персонализированная лента, НЕ хронологическая. Алгоритм ранжирует по множеству сигналов.

| Сигнал | Вес | Описание |
| --- | --- | --- |
| Близость пользователь-контент | ~30% | Как часто юзер взаимодействует с автором |
| Скорость ранней вовлечённости | ~20% | Лайки/комменты/репосты в первые 30-60 мин |
| Свежесть контента | ~15% | Свежесть (полураспад 4-6 ч для текста, дольше для видео) |
| Предпочтение типа контента | ~10% | Предпочтение юзером видео/текста/картинок |
| Социальное доказательство | ~10% | Общее кол-во реакций / размер аудитории |
| Авторитет автора | ~10% | Возраст аккаунта, стабильность, ER |
| Негативные сигналы | ~5% | "Скрыть", отписки, жалобы |

## 2. VK Clips — главный рычаг роста

VK Clips — короткие видео (аналог TikTok). ~700-900M ежедневных просмотров на платформе. Стратегический приоритет VK 2025-2026. Отдельный алгоритм от основной ленты.

#### Алгоритм обнаружения Clips

```
1. Новый Clip → тест на ~200-500 юзеров
2. Процент досмотра + вовлечённость > порога → расширение до ~2,000-5,000
3. Каждый раунд оценивает: досмотр, лайки, комменты, репосты, визиты профиля
4. Топ-Clips → 100K-1M+ просмотров НЕЗАВИСИМО от числа подписчиков
5. Срок жизни: недели (vs часы для постов ленты)
```

| Параметр | Рекомендация |
| --- | --- |
| Длина | 15-45 секоптимально. Макс 180 сек. До 60 сек — лучший процент досмотра |
| Формат | 9:16вертикальный (мобильный приоритет) |
| Разрешение | 1080x1920 минимум |
| Хук | Первые 1-3 секунды ОБЯЗАН захватить внимание |
| Субтитры | ВСЕГДА добавлять — +15-25% к проценту досмотра |
| Музыка | VK музыка (без проблем с авторскими правами). Трендовые звуки = буст |
| Хэштеги | 3-5: микс из широких (#карьера #работа) и нишевых (#AI #нейросети) |

#### Форматы Clips для AI/Карьера ниши

1. "А вы знали?" факты15-30 сек. "ChatGPT экономит 3 часа в неделю. Вот как." Высокий досмотр (цикл любопытства).

2. Трансформации До/После"Резюме до AI vs после AI-оптимизации." Визуально, удовлетворяющий, расшариваемый.

3. Быстрые советы/лайфхаки (списки)"3 промпта которые изменят ваше сопроводительное." Нумерованные списки отлично работают.

4. Реакция на новостиКомментарий к трендовым карьерным/AI новостям. Использует тренды для обнаружения.

5. День из жизни"Что делает AI-консультант за день." Вдохновляющий контент, строит личный бренд.

6. Разрушение мифов"Нет, AI не заменит вас. Но тот, кто использует AI — заменит." Противоречие стимулирует вовлечённость.

7. Мини-туториал (запись экрана)Запись экрана с демо AI инструмента. Образовательный + практический формат.

8. Проверка реальных зарплат"Реальные зарплаты [роль] в Москве/СПб/Удалённо." Данные = вовлечённость.

> **Инсайт:** Clips = Машина роста для 17 подписчиков.Охват Clips = 200-1000%+ от базы подписчиков. С 17 подписчиков один Clip может набрать тысячи просмотров через ленту обнаружения. Рекомендация: 1-2 Clips/день первые 30 дней.

#### Хук-формулы для VK Clips

```
"Вы совершаете эту карьерную ошибку прямо сейчас..."
"AI-инструмент о котором не знают 99% профессионалов"
"HR-менеджер здесь. Прекратите делать это в резюме."
"Я тестировал [AI tool] 30 дней. Вот что случилось."
Числа в хуке: "3 способа...", "5 ошибок...", "прибавка ₽Х к зарплате..."
```

> **Внимание:** Перезалив без водяных знаков!VK штрафует контент с водяными знаками TikTok. Перерендерите нативно перед загрузкой.
