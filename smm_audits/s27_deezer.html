1. Алгоритм и обнаружение

Рекомендательный движок Flow

Deezer Flow — AI-движок, изначально для музыки, расширен на подкасты с 2020. Гибрид коллаборативной фильтрации + контентного анализа + редакционной курации.

СигналОписание
Процент дослушиванияЭпизоды, прослушанные на 75%+ = сильный позитивный сигнал. Под 30% = плохое совпадение
Холодный стартДля новых шоу (СБОРКА = 4 эпизода) алгоритм полагается на метаданные, а не коллаборативную фильтрацию
Подписки/ОтпискиДинамика подписок — сигнал интереса
Повторные прослушиванияПовторные прослушивания = высокое качество контента
Время сутокАлгоритм учитывает привычное время прослушивания пользователя
10-15 преданных слушателей, которые дослушивают эпизоды, могут запустить алгоритмическое продвижение. Критическая масса для подкаста на Deezer очень низкая.

Поиск

  • Вес заголовка = #1. "СБОРКА" — уникальное слово с низкой конкуренцией в глобальном индексе Deezer
  • Описание полностью индексируется. Русский текст поддерживается
  • Точное совпадение > нечёткое. "Сборка подкаст" находится сразу, транслитерация "sborka" — ниже
  • Буст свежести: шоу с недавним эпизодом ранжируется выше

Редакционная курация

Deezer активнее в ручной курации подкастов, чем Spotify. Редакция ищет не-англоязычный, не-французский контент для диверсификации каталога. Контакт: podcasts@deezer.com

Гео-персонализация

Deezer персонализирует контент по локации и языку. Пользователи в Германии/Франции/Израиле с русскоязычными настройками увидят русские подкасты приоритетнее. Кросс-языковое обнаружение: если RU-пользователь слушает EN-карьерные подкасты, Deezer может рекомендовать СБОРКУ по тематике.

Факторы ранжирования в категориях

  1. Регулярность побеждает: Еженедельное обновление > спорадическое
  2. Количество эпизодов: 20+ эпизодов = попадание в "Популярное". СБОРКА пока 4 → цель: 10-12
  3. Скорость подписок: 20-30 новых за неделю = Трендовый в категории
  4. Дослушиваемость — главное: 14-мин формат СБОРКИ идеален
  5. Редакционный отбор: Не-EN шоу недопредставлены → питч вероятнее

Бесплатный vs Премиум

  • Бесплатный: Стриминг подкастов без ограничений. Реклама между треками (не подкастами)
  • Премиум (10.99 EUR/мес): Оффлайн, Hi-Fi, без рекламы
  • Для СБОРКИ: Подкасты доступны бесплатно всем пользователям = максимальный охват

Социальные функции

  • Поделиться ссылкой на эпизод → получатель получает уведомление, если на Deezer
  • Шеринг в соцсетях усиливает обнаружение (вирусная петля)
  • "Потому что вы слушали..." — рекомендации после завершения эпизода
  • "Трендовое в [Категории]" — скорость роста подписчиков + скорость прослушиваний
