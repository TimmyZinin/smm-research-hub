Data-driven инфографика — их топ-формат. Для СБОРКИ: нельзя конкурировать по объёму данных, но можно по actionability: "HH говорит 30% резюме отклоняют за 6 сек. Вот что изменить (пошагово)".

@tcareer — T-Career (Тинькофф)

Expert authority + bold takes = рост в карьерной нише. Для СБОРКИ: Тим не должен бояться спорных карьерных мнений. "Unpopular opinion: cover letters are dead" — 2-3x лучше generic advice.

4. Тон коммуникации для бренд-канала

ДЕЛАТЬ

  • "Мы" / "Команда СБОРКИ" для общих постов
  • "Тим Зинин, основатель" для экспертных мнений
  • Структурированные списки и чек-листы
  • Цифры и данные (даже от 55 участников)
  • Конкретные примеры и шаблоны
  • Спорные карьерные мнения (2-3x вовлечённости)
  • Эмодзи минимально: 1-2 на пост, для структуры
  • Русский язык, допустим IT-сленг

НЕ ДЕЛАТЬ

  • Шаблонная мотивация («верь в себя!») — max 1/неделю
  • Длинные абзацы без структуры
  • Промо больше 15% от всех постов
  • Спам-рассылки (причина блокировок на вебинаре #3)
  • Копирование формата HH/Habr — у них data, у вас system
  • Ненужные англицизмы в карьерных советах
  • Публикации в пятницу вечером (самый низкий % просмотров)
