1. Алгоритм и обнаружение

Яндекс.Музыка уникальна интеграцией в экосистему Яндекса: поиск, Дзен, Алиса, Яндекс.Станция. Это создаёт каналы обнаружения, недоступные ни одной другой подкаст-платформе.

Формула ранжирования в поиске

Оценка релевантности = (Совпадение заголовка x 3.0) + (Совпадение описания x 1.5) + (Совпадение заголовка эпизода x 1.0) + (Совпадение категории x 0.5) + (Бонус популярности x 0.3)

Алиса и голосовой поиск

20M+ устройств с Алисой (Яндекс.Станция, Браузер, Авто, приложение). Голосовые команды — мощный канал обнаружения.

Оптимизация для голосовых запросов:

  • "Алиса, включи подкаст Сборка" — прямой запрос (работает при уникальном имени)
  • "Алиса, включи подкаст про карьеру" — тематический запрос (нужна правильная категория)
  • "Алиса, включи подкаст про поиск работы" — ключевое слово в описании
  • "Алиса, что послушать про собеседование" — открытое обнаружение

"Сборка" — идеальное имя для Алисы: короткое, фонетически чёткое, без двусмысленности.

"Горячие подкасты" и подборки

Редакция Яндекс.Музыки создаёт тематические подборки ("Подкасты для карьеры", "Подкасты для саморазвития"). Попадание = эквивалент Spotify "New & Noteworthy". Триггеры: растущая аудитория + регулярные выпуски + высокий процент дослушивания.

Техническая оптимизация аудио

  • Формат: MP3, 128-192 kbps CBR
  • Частота дискретизации: 44.1 kHz
  • Каналы: Моно (для голоса — меньше размер, стабильная громкость)
  • Громкость: -16 LUFS (стандарт подкастинга)
  • Тишина в начале: Не более 1 секунды (быстрый старт = низкий процент пропусков)
  • Текущие эпизоды СБОРКИ (12.5-12.6 МБ) — в оптимальном диапазоне

5. Тон коммуникации для аудио

Специфика русскоязычного подкаст-аудио

ПараметрРекомендация
Темп речи140-160 слов/мин. Ускорять на волнующих моментах, замедлять на ключевых тезисах
Обращение"Ты", не "Вы" — последовательно с текущим подходом
ЛексикаБез академизмов: "этот" (не "данный"), "делать" (не "осуществлять")
Коллоквиализмы"Короче", "ну вот" — умеренно, для естественности
РегиональностьСтандартное московское произношение, без региональных маркеров
ЮморСухой, самоироничный. Без натужных шуток

Культурные ожидания RU-аудитории

  • Прямота ценится. "К сути быстро" — 30-секундное интро перед хуком = правильно
  • Личные истории резонируют: "Я сам когда-то..." — мощный хук
  • Избегать токсичного позитива. Русская аудитория не доверяет безудержному оптимизму. "Тёплый, поддерживающий, но реалистичный" = идеально
  • Экспертность важна: "Мой опыт", "я видел сотни кейсов" — строит доверие

Оптимизация для TTS (ElevenLabs)

  • Избегать сокращений: "то есть" (не "т.е.")
  • Числа прописью: "двадцать пять" (не "25")
  • CAPS для акцентов: "И вот это ГЛАВНОЕ"
  • Запятые = точки дыхания для TTS
  • Иностранные слова: транслитерация или пояснение
