1. Алгоритм и обнаружение

1.1 Как Spotify рекомендует подкасты (2025-2026)

Три ключевых системы: Collaborative Filtering (основной сигнал — слушатели похожих подкастов), Content-Based Signals (NLP title/description + автоматическая транскрипция + категории), Engagement Metrics.

1.4 Search Ranking

  1. Exact match в заголовке шоу — максимальный вес. "СБОРКА" → #1 для "сборка подкаст"
  2. Keywords в описании — второй вес
  3. Keywords в заголовках эпизодов — индексируются
  4. Автоматические транскрипты — с 2025 Spotify индексирует аудио
  5. Engagement — высоко-вовлечённые шоу ранжируются выше
  6. Свежесть — недавно обновлённые шоу → выше
Триггеры для editorial attention: Публикация consistently, >50% avg completion, >10% w/w рост подписчиков, упоминания в соцсетях со Spotify-ссылками.

2.2 Video Podcasts на Spotify

Video = 2-3x больше impressions чем audio-only. Видео-эпизоды показываются с визуальными thumbnails в ленте (значительно выше CTR).
ПараметрТребование
РазрешениеМин. 1080p (1920×1080), 16:9
ФорматMP4 (H.264 video, AAC audio)
ЗагрузкаЧерез Spotify for Podcasters (не RSS)
Макс. размер4GB / эпизод
СубтитрыАвто-генерация, лучше ручная (SRT)

Стратегия для СБОРКИ: Presentation-style (слайды + voiceover) — просто в производстве через ffmpeg. Видео shorts из Lisa pipeline можно репурпозить.

2.3 Интерактивные функции

  • Polls — карточки в плеере. Буст engagement. "Что для тебя самое сложное в поиске работы?"
  • Q&A — слушатели задают вопросы через страницу эпизода → контент для будущих эпизодов
  • Chapters — список секций в плеере. Снижает abandonment
  • Transcripts — индексируются для поиска. Загрузить ручные для точности на русском

2.4 Checklist на каждый эпизод

  • ☐ Заголовок содержит primary keyword в первых 5 словах
  • ☐ Описание включает 3-5 keywords естественно
  • ☐ Chapter markers проставлены (intro, content, tips, CTA)
  • ☐ Транскрипт загружен (из скрипта)
  • ☐ Poll или Q&A добавлен
  • ☐ Ссылки в show notes с UTM
  • ☐ "Follow" CTA произнесён на 2-3 минуте и в конце

3.3 Конкурентные преимущества СБОРКИ

  • AI angle — мало карьерных подкастов про AI в поиске работы
  • Короткий формат — 14-15 мин идеален для completion rate (конкуренты 30+ мин)

3.4 Gaps to Exploit

  1. Video podcasts — почти никто из RU-карьерных не делает видео на Spotify
  2. AI job search tools — никто не учит практическому AI-для-карьеры в RU-подкастах
  3. Mini-episodes — нет RU-карьерных подкастов с 3-5 мин quick tips
  4. Interactive features — polls, Q&A практически не используются
  5. Social clips — RU-подкастеры редко создают short clips для соцсетей

5. Tone of Voice для Spotify

5.1 Audio Style Guide

  • Opening hook (первые 15 сек) — CRUCIAL. НЕ "Привет, это Сборка" как первые слова
  • Паттерн: [HOOK] → [JINGLE 5 сек] → [Standard intro] → [Content]
  • Пример: "Ты отправил 50 откликов и ни одного ответа. Знакомо? Сегодня разберёмся, почему." [JINGLE] "Привет, это Сборка..."
