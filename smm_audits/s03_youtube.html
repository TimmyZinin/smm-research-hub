4. Tone of Voice — Стиль для YouTube

Позиционирование голоса

Стиль Тима на YouTube

  • Нарративный — история первична, урок вторичен
  • Show, don't tell — B-roll + визуальный storytelling
  • Документальное спокойствие — авторитет, а не hyperactive energy
  • Conversational — разговор С зрителем, не лекция
  • Vocal variety — паузы перед ключевыми утверждениями
  • Тонкая фоновая музыка — задаёт эмоциональный тон

Чего НЕ делать

  • НЕ копировать MrBeast-стиль гиперактивности
  • НЕ быть монотонным — vocal variety matters
  • НЕ начинать с тишины или fade-in
  • НЕ лекции-монологи без визуала
  • НЕ более 3 CTA за видео
  • НЕ кликбейт без delivery

Структура скрипта (8-12 мин видео)

[0:00-0:30] HOOK — Самый сильный момент/утверждение/визуал [0:30-1:30] CONTEXT — Почему это важно + кто ты (для новых зрителей) [1:30-3:00] STORY/SETUP — Нарратив, который ведёт видео [3:00-6:00] CORE CONTENT — Основная ценность (советы, инсайты, анализ) [6:00-7:00] CTA #1 — Мягкий CTA (sborka.work, подписка) [7:00-9:00] DEEPER INSIGHT — "One more thing" для тех кто досмотрел [9:00-10:00] CONCLUSION + CTA #2 — Жёсткий CTA (подписка + клуб) [10:00-10:30] END SCREEN — Визуальные CTA на следующее видео/плейлист

5 формул хуков для Тима

1. The Reveal Hook
Начать с самого удивительного момента видео, потом cut: "3 дня назад..." — используя существующий travel-footage.
2. The Question Hook
"Что будешь делать, когда AI заменит 80% твоей работы? Я реально это проверил." — провокация + personal proof.
3. The Location Hook
Stunning визуал города/места + voiceover: "Я приехал сюда, чтобы найти кое-что." — сила travel-контента.
4. The Contradiction Hook
"Все говорят [распространённое мнение]. Я обнаружил обратное." — работает для карьерных и AI-тем.
5. The Personal Hook
"Это видео чуть не осталось неснятым. Вот почему." — уязвимость + интрига.

Рекомендация для Тима: 30-45 секунд. Документальный/нарративный стиль требует времени для микро-истории, но достаточно коротко для высокого completion rate. Формат 9:16 (vertical) обязателен.

Shorts → Long-form Pipeline

Жёсткая правда: Shorts-подписчики НЕ надёжно смотрят long-form. Конверсия: 0.5-2% от views Shorts. Short с 100K просмотрами = 500-2,000 views на long-form. Но это работает через узнаваемость бренда.
Teaser Shorts
Нарезать из long-form. Закончить: "Полное видео на канале" (НЕ "ссылка в био" — YouTube депрайоритизирует external redirects). Pin комментарий со ссылкой.
Series Shorts
3-5 частей серии, наращивающих curiosity. Часть 5: "Я снял об этом полную документалку". Серии: "30-day AI challenge", "AI tool за 60 секунд", "Career tip дня".

7. Growth Hacks — Нестандартные тактики

Reactivation Sequence (перед comeback-видео)
2 недели подготовки = +40% к engagement первого видео
Неделя -2: Community poll "Какой контент хотите?". Неделя -1: Image teaser behind-the-scenes. Launch day: Text post + видео. Неделя +1: Клип из видео.
Cross-Platform Funnel: TG → YouTube
5-15% конверсия из TG-канала
Тизер (первые 30 сек или ключевой клип) в TG канал. "Полное видео на YouTube" + прямая ссылка. Polls в TG для генерации идей (зрители чувствуют причастность).
Cross-Platform Funnel: LinkedIn → YouTube
2-8% конверсия из поста LinkedIn
LinkedIn пост с key insight из видео. Дать ценность в самом посте (LinkedIn любит native). Внизу: "Снял полное видео: [ссылка]".
Premiere Strategy для event-контента
Live chat при Premiere = engagement boost для алгоритма
Запланировать Premiere за 24-48ч. Промо во всех каналах. Использовать для comeback-видео и документалок. НЕ для обычных загрузок.
Comment-First Hour
Reply на КАЖДЫЙ комментарий в первый час = algorithmic boost
Быть онлайн в момент публикации. Heart + reply на каждый комментарий. Pin CTA-комментарий (sborka.work + следующее видео). Задать вопрос В видео, чтобы спровоцировать комменты.
YouTube Podcasts Tab
Новая поверхность с низкой конкуренцией
Любые long-form разговоры/интервью дополнительно публиковать как "Podcast" в YouTube Studio. Появляется в Google Podcasts, YouTube Music и отдельной вкладке.
Collaboration Strategy
Guest appearances на каналах 2-5x больше = самый быстрый рост
Таргет: русскоязычные tech/career creators. Предложение: "Расшарю клип своей аудитории на всех платформах". Даже response videos работают. 1-2 коллаба/мес после Month 3.
Unfair advantage Тима: Кросс-платформенная аудитория (TG + LinkedIn + VK + Instagram). Большинство мелких YouTubers не имеют existing аудитории на других платформах. Это = built-in seed audience для каждого видео.
