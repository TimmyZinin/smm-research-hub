1. Fediverse Mechanics & Content Distribution

Mastodon is the largest platform in the Fediverse — a network of interconnected, independently operated servers communicating via the ActivityPub open protocol. ~10-12M active monthly users collectively, with Mastodon accounting for 5-7M. The user base is disproportionately developers, academics, and open-source advocates — exactly Tim's target audience.

Critical Difference: Mastodon has NO algorithm. Content discovery is purely chronological and follower/hashtag-driven. Boosts (retoots) are the ONLY viral mechanism. There is no «For You» page, no engagement optimization, no pay-to-play.

4. Community Culture & Accessibility

Mastodon's culture is shaped by anti-surveillance capitalism, open source ethos, and accessibility-first principles. Understanding this is non-negotiable for success.

Core Values:
  • Anti-surveillance capitalism — any «growth hacking» or «funnel» language = hostility
  • Open source & decentralization — FOSS contributions earn massive respect
  • Accessibility is not optional — alt text, CamelCase hashtags, content warnings = basic courtesy
  • Community over celebrity — nobody wants to be a «Mastodon influencer»
  • Anti-metrics obsession — celebrating follower counts is frowned upon

What Earns Respect vs Gets You Unfollowed

EARNS RESPECT

  • Sharing open-source tools/code
  • Writing detailed alt text
  • Boosting others' content
  • Admitting mistakes publicly
  • Using CW appropriately
  • Responding to questions helpfully
  • Contributing to Fediverse projects

GETS YOU UNFOLLOWED

  • No alt text on images consistently
  • Cross-posting with «RT @user» format
  • Excessive self-promotion
  • Corporate/marketing language
  • One-way broadcast behavior
  • Mentioning follower counts
  • «Follow me on Twitter/X» links

Accessibility Requirements (NON-NEGOTIABLE)

Alt Text: EVERY image must have alt text. For code screenshots: describe the code. For infographics: describe key data points. For architecture diagrams: describe components and connections. Mastodon UI warns if you post without alt text. Some clients hide images without it. auto_publisher.py MUST include alt text in every media upload.
CamelCase Hashtags: #ArtificialIntelligence, NOT #artificialintelligence. Screen readers parse CamelCase as separate words. Violating this signals «outsider.»

5. Hashtag Strategy

Hashtags are the primary discovery mechanism on Mastodon. Users can FOLLOW hashtags (since 4.x), making them appear in Home timeline. Use 3-5 per post, max 7. More = spam.

DO

  • Write as practitioner sharing experience
  • First person: «I built,» «I learned»
  • Be specific: numbers, code, architecture
  • Acknowledge uncertainty: «not sure this scales, but...»
  • Show the messy middle, not polished results
  • Reference open-source tools, credit builders
  • Dry, self-deprecating humor

DON'T

  • LinkedIn-speak: «Excited to share,» «Thrilled to announce»
  • Marketing: «revolutionary,» «disrupting,» «10x»
  • Growth-hacker: «funnel,» «conversion,» «lead magnet»
  • CTA urgency: «Don't miss out,» «Limited time»
  • Cross-post without tone adaptation
  • Emoji walls
