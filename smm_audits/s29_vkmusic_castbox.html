1. Алгоритм и обнаружение

VK Music

Социальный граф = #1 сигнал. Если друг ВК слушает подкаст — он появляется в рекомендациях с соцпруфом ("3 друга слушают").

  • Лента "Для тебя": подкасты рядом с музыкой
  • Интеграция с сообществами: участники карьерных сообществ → карьерные подкасты
  • Буст свежести 24-48ч для новых эпизодов
  • ~12% рынка RU подкастов

CastBox

Рекомендации на глубоком обучении. Анализирует историю прослушиваний, паттерны пропусков, дослушиваемость, оценки.

  • Трендовые списки по категориям
  • Комментарии к аудио = сигналы вовлечения
  • 2-4 недели буст обнаружения для новых шоу
  • 30M+ пользователей, силён на Android
VK Music: каждый слушатель-VK-юзер создаёт волну через сеть друзей. Социальный граф > алгоритм.
CastBox: RU-карьерных подкастов практически нет. Комментарии к аудио + клипы = уникальные инструменты вовлечения.

Конкурентные преимущества СБОРКИ

  1. 14 мин — короче всех. Соответствует молодой аудитории VK + мобильной ориентации
  2. Тёплый тон vs. корпоративный/HR-ориентированный у конкурентов
  3. Продуктовая воронка: бот + сообщество + клуб (конкуренты = отдельные медиа)
  4. Стабильность через AI: регулярная публикация без ограничений расписания
  5. Низкая конкуренция: каталог подкастов VK Music меньше Spotify/Apple
CastBox: русскоязычных карьерных подкастов практически нет. Диаспорная аудитория на Android = недообслуженный рынок.

4. Процесс подачи

VK Music (10 шагов)

  1. Перейти на vk.com/app51488920
  2. Авторизоваться через VK
  3. Выбрать "Подключить подкаст"
  4. Вставить RSS: cloud.mave.digital/69800
  5. Подтвердить владение (email)
  6. Заполнить категорию: "Образование"
  7. Загрузить обложку 3000x3000
  8. Отправить на модерацию (1-3 дня)
  9. При капче — повторить через VPN
  10. После одобрения — RSS-синхронизация автоматическая

CastBox (11 шагов)

  1. creators.castbox.fm → Регистрация
  2. Войти через Google
  3. Импорт подкаста → RSS URL
  4. Вставить: cloud.mave.digital/69800
  5. Предпросмотр метаданных → проверить
  6. Подтверждение email (tim.zinin@gmail.com)
  7. Подтвердить владение
  8. Указать категории: Careers + Education
  9. Указать язык: Русский
  10. Отправить на проверку (24-48ч)
  11. После одобрения: автоматическая RSS-синхронизация

Поиск VK Music

  • Заголовки с ключевыми словами: "Сборка | Карьера, резюме, работа мечты"
  • Хэштеги в VK-постах: #карьера #работа #резюме #собеседование #сборка
