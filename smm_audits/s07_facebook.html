2. Three-Tier Strategy: Profile + Page + Group

Profile (Primary)

Роль: Основной контент-хаб, 5-10x reach vs Page

Формат: 1 пост/день, 80% value / 20% promo

Professional Mode: Включить НЕМЕДЛЕННО — Follow button, Reels bonus, analytics

Лимит: 5,000 друзей (место для 3,400 новых)

Page (Infrastructure)

Роль: Ads platform, Events хостинг, IG auto-crosspost

Формат: Минимальный контент, НЕ дублировать Profile

Когда: Создать, но не тратить время на organic

Для: Платный трафик когда готов

Group (Community)

Роль: Highest engagement (10-30x vs Page), conversion engine

Название: "AI-Powered Job Search (СБОРКА Community)"

Когда: Запуск в неделю 1, рост до 1,000 за 3 мес

Content: 80% value / 15% engagement / 5% promo

НЕ конвертировать Profile в Page. В 2026 Profile получает 50%+ больше reach. Professional Mode даёт все нужные фичи (followers, analytics, scheduling).

5. Tone of Voice для Facebook

РАБОТАЕТ

  • Личные истории с lessons learned
  • Документальный стиль ("я построил X, вот что узнал")
  • Saves-worthy контент (чеклисты, шаблоны, фреймворки)
  • Social proof (80-120 shares на 1K views — 2-3x лучше generic tips)
  • Вопросы в конце поста (триггерят comments = буст)
  • Bilingual: RU для community, EN для Reels reach

НЕ РАБОТАЕТ

  • Direct link posts (50% lower reach)
  • Generic мотивация без конкретики
  • Overposting (10+/день = убивает engagement)
  • Игнорирование комментов (breaks community signal)
  • Продажа в каждом посте (audience unfollows)
  • Inconsistent posting (алгоритм deprioritizes)
